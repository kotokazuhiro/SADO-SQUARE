# SADO SQUARE
​
## サイト概要
### サイトテーマ
佐渡島の美味しいお店や綺麗な景色、子供が安心して遊べる場所などを検索したり、
自分が気に入っているとこを投稿したりできるコミュニティサイト
​
### テーマを選んだ理由
大学進学を機に、佐渡島から離れ、数年ぶりにまた佐渡島で生活するようになり、
当時では、気がつく事ができなかった佐渡島の魅力に気づいたので、
生まれ育った場所を、みんなに知ってもらいたいという思いからテーマを選びました。
2023年から世界遺産登録に向けて、動き出しております。
そんな佐渡島のさらなる発展のため、全国や世界の人々に、知ってもらいたい。
住んでいる人も観光で訪れた人達もよかったことを共有し、佐渡島の発展につながってほしいと思いました。
こちらのコミュニティサイトを利用することによって、佐渡を初めて訪れる人はどこに行ったら良いのか、
こんなところが人気であるという事がわかり、観光の予定を組みやすくなるのではと考えます。
また佐渡では、移住を考えている方のために、半年間住んでみる事ができる仕組みがあります。
佐渡でしかない魅力を感じたり、体験することができる、その仕組みを活用して頂くための第一歩になるのではと考えます。

​
### ターゲットユーザ
- 佐渡島を訪れたいがどこにいったらいいのかわからないと思っている人
- 佐渡島の魅力を知りたい人
- 年齢別に何が人気になっているか知りたい方
- 新商品(食や施設など)の開発、製造に為の参考として
​
### 主な利用シーン
- 佐渡島のおすすめを検索する時
- 訪れたところの感想を見たい時
- 自分が行ったところで感じたことを投稿したい時
- 最近の佐渡島では、どんなものが人気なのかを知りたいに検索する時
